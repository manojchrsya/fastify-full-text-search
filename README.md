## Implementing similar to Trie algo to search json object
--------------------------------------------------------------
### Installation

It requires [Node.js](https://nodejs.org/) to run.
Install the dependencies and devDependencies and start the server.

```sh
$ cd project-dir
$ npm install -d
$ npm run dev or npm run start
```
